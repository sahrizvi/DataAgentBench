You are working with two datasets:

1. **review_dataset**
- Contains user reviews for Google Maps locations.
- Fields:
    - name (str)
    - time (str)
    - rating (int)
    - text (str)
    - gmap_id (str) 

2. **meta_dataset**
- Contains business metadata for Google Maps locations.
- Fields:
    - name (str)
    - gmap_id (str)
    - description (str)
    - num_of_reviews (int)
    - hours (list)
    - MISC (dict)
    - state (str)
