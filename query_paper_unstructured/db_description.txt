You are working with two databases to solve this query.

Here are the descriptions of these two databases:

1. paper_docs_database
   - This database is stored in a MongoDB database and contains unstructured text documents about academic papers from HCI research venues.
   - This database consists of one collection:
     - paper_docs
       - This collection contains full text content of academic papers. Each document represents a single paper with its complete text.
       - Fields:
         - _id: Unique document identifier
         - filename (str): Name of the source file (e.g., "A Lived Informatics Model of Personal Informatics.txt")
         - text (str): Full text content of the paper

   - Each document contains the full text of an academic paper. For each paper, you may need to extract:
     - title: The title of the paper (matches the filename without .txt extension)
     - year: Publication year
     - venue: Conference or journal venue (e.g., "CHI", "Ubicomp", "CSCW")
     - source: Publisher (e.g., "ACM", "IEEE")
     - domain: Research domain/topic (e.g., "food", "physical activity", "sleep", "mental")
     - contribution: Type of contribution (e.g., "empirical", "artifact", "theoretical", "survey")

2. citations_database
   - This database is stored in a SQLite database and contains structured citation information for papers.
   - This database consists of one table:
     - Citations
       - This table contains citation records for papers by year. Paper titles can be joined with paper information extracted from the paper documents.
       - Fields:
         - id (int): Unique identifier for the citation record
         - title (str): Title of the paper (joining key with paper_docs)
         - citation_count (int): Number of citations received in that year
         - citation_year (int): Year the citations were received

To answer queries, you will typically need to:
1. Query the Citations table from the SQLite database
2. Read the paper documents from MongoDB to extract paper information (domain, venue, source, contribution, year)
3. Join the citation data with the extracted paper information based on title
