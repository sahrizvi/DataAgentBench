The agent violates explicit system-level or tool-level usage constraints in system prompt and tool specification, despite those constraints being clearly specified prior to execution, resulting in incorrect or failed behavior. 

This FM includes: 
- Incorrect formatting required by tool interfaces 
- Violations of execution environment constraints 
- Failure to escape, quote, or encode content as explicitly required

Necessary & sufficient conditions of this FM:
- The system or tool specification explicitly states a constraint
- The agent’s output violates that constraint
- The failure occurs independently of task logic correctness

This FM excludes:
- Logic errors when the agent followed the interface correctly
- Errors due to misunderstanding the task objective
- Errors due to missing data knowledge
